Regional Leadership Hub (iOS)

About this app

Regional Leadership Hub is an internal Discount Tire Region 2 (Tennessee) leadership tool. It is distributed only to authorized regional leaders via TestFlight or direct internal install. If you received this app and are not an authorized Region 2 leader, please do not use it.

How to get help

For questions, access requests, bug reports, feature requests, or data correction requests, contact the regional administrator:

Email: thevictorvogel@gmail.com

When reporting a bug, please include:

  • What you were trying to do
  • What happened instead
  • iOS version and device model
  • App version (visible in the Admin or Settings area)
  • A screenshot if it helps explain the problem

Common questions

How do I get access?

Access is by invitation only. The regional administrator adds authorized leaders to the regional roster and to the TestFlight tester group. If you should have access and don’t, email the address above.

I can’t sign in

Confirm you are using the email address on file with the regional roster. If sign-in still fails, email the regional administrator and include the error message and the time of the attempt.

I see “demo” data instead of real data

The app supports a training/demo mode. Switch to the live regional backend in Admin settings, or ask the regional administrator to walk you through the connect-to-live step.

How is my data handled?

See the Privacy Policy for what is collected, where it is stored, and who can see it.

Status and known issues

This app is in active internal beta. Known limitations and in-progress work are tracked privately by the regional team. If something looks broken, please report it via the support email so it can be triaged.
